# C10F004B 收发干扰手动测试指南

---

## 一、这个测试是干什么的？

模块里面同时有发射通道和接收通道，两个通道靠得很近。发射通道工作时功率很大（2W 甚至 3W），如果它的信号"泄漏"到了接收通道，就会干扰接收——相当于你在听一个人小声说话，旁边有人在大喊大叫。

这个测试就是检查：发射通道工作时，接收通道有没有被干扰。

---

## 二、需要什么设备？

| 设备 | 用途 |
|------|------|
| 信号源（能输出 1.05~1.55 GHz 正弦波） | 模拟发射中频信号 |
| 频谱仪（能看到 0.85~1.65 GHz 范围） | 观察接收通道噪底 |
| 合路器 ×2 | 把 10M 参考和信号合并到一起 |
| 10 MHz 参考信号源 | 给模块提供参考时钟 |
| DC 电源 ×2 | 给模块 RX（12V）和 TX（24V）供电 |
| Bias-Tee ×2（或偏置器） | 分离 DC 电和 RF 信号 |
| 同轴电缆若干 | 连接各个设备 |
| 50Ω 匹配负载（堵头） | 堵住 RF 端口 |

---

## 三、怎么接线？

一步一步来，先接好所有线缆，最后再通电。

### 第一步：接发射通道（TX IF 口）

```
信号源 ──→ 合路器 RF 口
10M 参考 ──→ 合路器 10M 口
合路器 输出 ──→ Bias-Tee RF 口
DC 电源 24V ──→ Bias-Tee DC 口
Bias-Tee RF+DC 口 ──→ 模块 TX IF 口
```

> 信号源先设置输出 1.05 GHz，功率 −14 dBm，**RF 先关掉不要开**。

### 第二步：接接收通道（RX IF 口）

```
频谱仪输入 ←── Bias-Tee RF 口
10M 参考 ──→ 合路器 10M 口
合路器 输出 ──→ Bias-Tee DC? 口      ← 注意：这里 10M 从频谱仪那边馈入
DC 电源 12V ──→ Bias-Tee DC 口
Bias-Tee RF+DC 口 ──→ 模块 RX IF 口
```

> 另一路 10M 参考通过合路器从频谱仪端馈入 RX IF 口。频谱仪本身不需要 10M，这里只是借道把 10M 送进模块。

### 第三步：RF 端口处理

| 端口 | 接什么 |
|------|------|
| RF IN（接收射频输入） | 接 **50Ω 匹配负载**（堵头），不接天线 |
| RF OUT（发射射频输出） | **不接任何东西**（空着），或者也接一个堵头 |

### 第四步：检查接线

对着模块看一下四个口：

- TX IF 口 → 连着 Bias-Tee → 连着信号源（24V 供电）
- RX IF 口 → 连着 Bias-Tee → 连着频谱仪（12V 供电）
- RF OUT → 空着或堵头
- RF IN → 堵头

确认无误后，**打开 10M 参考信号**，观察模块指示灯是否常亮（常亮 = 锁定正常）。

---

## 四、怎么测？

### 4.1 设置频谱仪

| 参数 | 设置值 | 为什么这么设 |
|------|:---:|------|
| 起始频率 | 0.85 GHz | 我们要看的 IF 范围是 0.95~1.55 GHz，留点余量 |
| 终止频率 | 1.65 GHz | 同上 |
| 参考电平 | −40 dBm 左右 | 噪底大概在 −60 左右，设太高看不到 |

> 频谱仪上应该能看到一条比较平的噪底线，大概在 −60 dBm 左右。这就是接收通道的背景噪声。

### 4.2 未开发射时检查噪底

1. 先只给 RX 上电（12V），TX 先不上电
2. 确认频谱仪上噪底在 −60 dBm 左右
3. 给 TX 上电（24V），但**信号源的 RF 输出保持关闭**
4. 观察频谱仪：噪底应该和之前差不多，**没有明显的尖刺或鼓包**

> 噪底抬高是正常的（TX 上电后内部电路工作会有一些噪声），但关键是**不能有尖刺**——尖刺就是发射通道泄漏过来的干扰信号。

### 4.3 开发射信号，手动扫频

这是最关键的一步。

1. 信号源设置：频率 1.05 GHz，功率 −14 dBm，**打开 RF 输出**
2. 观察频谱仪上 1.05 GHz 附近有没有尖刺或明显鼓包
3. 用手拧信号源的旋钮（或按上下键），慢慢把频率从 1.05 GHz 往 1.55 GHz 调
4. 眼睛盯着频谱仪屏幕，频率每变一点就停一下，看对应位置有没有异常
5. 一直调到 1.55 GHz 为止

> 怎么算"异常"？如果某个频率处，噪底突然冒出一个比周围高 2~3 dB 以上的尖峰，那就是发射信号泄漏到接收通道了。

---

## 五、怎么判定结果？

| 现象 | 结论 |
|------|------|
| 噪底全程平坦，无尖刺 | ✅ 合格，收发隔离良好 |
| 某个频率出现尖刺，比噪底高很多 | ❌ 不合格，存在收发干扰 |

> 如果出现不合格，记录下尖刺出现的频率和幅度，反馈给厂家。

---

## 六、重要提醒

1. **先接好所有线缆，最后再通电。** 不要带电插拔。
2. **10M 参考信号必须先给**，确认指示灯常亮后再上 DC 电。指示灯闪烁 = 没有锁定，检查 10M 接线。
3. **RF OUT 空着的时候不要碰它**——开发射时 RF OUT 口有功率输出，虽然没接天线但接头处仍有电磁辐射。
4. 信号源功率不要超过 −10 dBm，功率太大会让噪底整体抬高，反而看不清尖刺。
5. 频谱仪的 RBW（分辨率带宽）建议设 10~30 KHz，太宽了尖刺会被平均掉看不到。
