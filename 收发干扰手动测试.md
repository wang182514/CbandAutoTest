# C10F004B 收发干扰手动测试

---

## 一、测试目的

检查发射通道工作时，接收通道是否受到干扰。收发隔离不足时，发射信号会泄漏到接收链路，导致接收噪底出现异常尖峰。

---

## 二、所需设备

| 设备 | 用途 |
|------|------|
| 信号源（1.05~1.55 GHz） | 模拟发射中频信号 |
| 频谱仪（0.85~1.65 GHz） | 观测接收通道噪底 |
| 10 MHz 参考信号源 | 模块时钟参考 |
| 合路器 ×2 | 合并 10M 参考与信号 |
| Bias-Tee ×2 | 分离 DC 供电与 RF 信号 |
| DC 电源 ×2 | RX（12V）/ TX（24V）供电 |
| 50Ω 匹配负载 | 堵 RF 端口 |
| 同轴电缆若干 | 设备互联 |

---

## 三、接线

### 发射通道（TX IF 口）

```
信号源 1.05GHz −14dBm ──→ 合路器 ──→ Bias-Tee ──→ 模块 TX IF 口
10M 参考 ──────────────→ 合路器
DC 24V ───────────────────────→ Bias-Tee
```

> 信号源先设好频率和功率，RF 输出暂**关闭**。

### 接收通道（RX IF 口）

```
频谱仪输入 ←── Bias-Tee ←── 合路器 ←── 10M 参考
DC 12V ──────→ Bias-Tee
Bias-Tee RF+DC ──→ 模块 RX IF 口
```

### RF 端口

| 端口 | 连接 |
|------|------|
| RF IN | 50Ω 匹配负载 |
| RF OUT | 50Ω 匹配负载（或不接） |

---

## 四、测试步骤

### 4.1 频谱仪设置

| 参数 | 设置 |
|------|------|
| 起始频率 | 0.85 GHz |
| 终止频率 | 1.65 GHz |
| 参考电平 | −40 dBm |
| RBW | 10~30 KHz |

### 4.2 未开发射时检查

1. 先开 10M 参考，确认模块指示灯常亮
2. 给 RX 上电（12V），TX 上电（24V）
3. 信号源 RF 保持关闭
4. 观察频谱仪噪底，应在 −60 dBm 左右，无明显尖刺

### 4.3 开发射扫频

1. 信号源打开 RF 输出（1.05 GHz / −14 dBm）
2. 频谱仪上观察 1.05 GHz 附近有无异常尖峰
3. 手动调节信号源频率，从 1.05 GHz 逐步调至 1.55 GHz
4. 全程观察频谱仪，记录出现尖刺的频率和幅度

---

## 五、判定

| 现象 | 结论 |
|------|------|
| 全程噪底平坦，无尖刺 | 合格 |
| 某频率出现明显尖峰 | 不合格，记录频率和幅度 |

---

## 六、注意事项

1. 先接好全部线缆，最后通电
2. 10M 参考须先于 DC 供电接入，确认指示灯常亮后再上电
3. RX IF（12V）与 TX IF（24V）供电电压不同，不可接反
4. 信号源输出功率不超过 −10 dBm
