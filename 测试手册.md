# C10F004B C波段射频模块 — 产品规格书

> 版本：1.0

---

## 一、这是什么产品？

C10F004B 是一个**射频变频模块**。它的作用是把频率较低的信号变成频率较高的信号，反过来也可以。

通俗理解：地面通信设备工作在 L 波段（大约 1 GHz），天线收发工作在 C 波段（大约 6 GHz），两者频率不同，没法直接连在一起。这个模块就是中间的"翻译官"——把 L 波段翻译成 C 波段发出去（发射），把 C 波段翻译成 L 波段收回来（接收）。

---

## 二、模块长什么样，四个接口分别干什么？

模块有 **四个 SMA 同轴接口**（SMA 是一种常见的螺纹拧紧式高频接头），分两组：

```
  ┌──────────────────────────────────┐
  │                                  │
  │  RX IF ◄───┐            ┌───────►│ RF OUT
  │  (接收中频) │            │(发射射频)│
  │            │  C10F004B  │        │
  │  TX IF ◄───┘  C波段射频  └───────►│ RF IN
  │  (发射中频)   模块       (接收射频) │
  │                                  │
  └──────────────────────────────────┘
```

### 接收通道（RX，Receive）

"接收"就是模块从天线方向收信号。

| 接口 | 接口旁刻字 | 方向 | 干什么用的 |
|------|-----------|:---:|------|
| RX IF | `IF IN / 12V / 10M` | 信号进 | 接收中频输入口。L 波段信号从这里进去。同时 12V 直流电和 10MHz 参考信号也从这里送入。 |
| RF IN | `RF IN` | 信号进 | 接收射频输入口。C 波段信号（从天线来的）从这里进入模块。 |

信号流向：**RF IN → 模块 → RX IF**（C 波段进 → 变频 → L 波段出）

### 发射通道（TX，Transmit）

"发射"就是模块往天线方向送信号。

| 接口 | 接口旁刻字 | 方向 | 干什么用的 |
|------|-----------|:---:|------|
| TX IF | `IF IN / 24V / 10M` | 信号进 | 发射中频输入口。L 波段信号从这里进去。同时 24V 直流电和 10MHz 参考信号也从这里送入。 |
| RF OUT | `RF OUT` | 信号出 | 发射射频输出口。变频后的 C 波段信号从这里输出。 |

信号流向：**TX IF → 模块 → RF OUT**（L 波段进 → 变频 → C 波段出）

### 两组接口对比速查

| | 接收通道 | 发射通道 |
|------|---------|---------|
| IF 口刻字 | `IF IN / 12V / 10M` | `IF IN / 24V / 10M` |
| IF 口供电 | **12V** | **24V** |
| RF 口刻字 | `RF IN` | `RF OUT` |
| RF 口方向 | 信号**进**模块 | 信号**出**模块 |

---

## 三、"同轴供电"是什么意思？

普通的电子模块通常需要接三根线：一根信号线、一根电源线、一根参考时钟线。

C10F004B 的 IF 接口把这三根线**合并到一根同轴电缆里**，这就是"同轴供电"。一根同轴电缆同时跑：

| 信号类型 | 干什么的 |
|----------|------|
| **DC 电源** | 给模块供电。接收用 12V，发射用 24V。没有电模块不工作。 |
| **10 MHz 参考信号** | 一个非常稳定的时钟信号。模块用它来校准内部振荡器，确保输出频率准确。就像给手表对时——没有参考信号，模块的频率会漂。 |
| **IF 中频信号** | 真正的通信信号。接收时从模块出来，发射时送进模块。 |

### 怎么把混在一起的信号分开？

需要一个叫 **Bias-Tee**（偏置器）的小器件。它有三个口：

```
DC 电源 ──► Bias-Tee DC口
信号源  ──► Bias-Tee RF口
            Bias-Tee RF+DC口 ──► 模块 IF口
```

Bias-Tee 的作用就是"分拣"——把 DC 电从 RF+DC 口分离到 DC 口，把 RF 信号从 RF+DC 口分离到 RF 口。

## 四、频率范围

### 接收通道

| 阶段 | 频率 |
|------|------|
| 从 RF IN 口**输入**的 C 波段信号 | 5.85 ~ 6.45 GHz |
| 从 RX IF 口**输出**的 L 波段信号 | 0.95 ~ 1.45 GHz |

> 模块的作用：把 6 GHz 左右的信号变成 1 GHz 左右。例如天线收到一个 6.10 GHz 的信号，经过模块后，从 RX IF 口出来变成了约 1.20 GHz。

### 发射通道

| 阶段 | 频率 |
|------|------|
| 从 TX IF 口**输入**的 L 波段信号 | 0.95 ~ 1.55 GHz |
| 从 RF OUT 口**输出**的 C 波段信号 | 5.85 ~ 6.45 GHz |

> 模块的作用：把 1 GHz 左右的信号变成 6 GHz 左右。例如你给 TX IF 口送一个 1.20 GHz 的信号，RF OUT 口会发出约 6.10 GHz 的信号。

### 接收和发射 IF 频率为什么不一样？

| 通道 | IF 频率范围 | 原因 |
|------|:---:|------|
| 接收 | 0.95 ~ 1.45 GHz | 接收只需覆盖 500 MHz 带宽（5.85→6.45 对应的 IF 范围） |
| 发射 | 0.95 ~ 1.55 GHz | 发射需要额外的余量，覆盖更宽的 IF 范围 |

## 五、接收通道性能参数

### 5.1 噪声系数（NF）

**噪声系数**是什么？任何电子器件自身都会产生一些随机的电噪声。如果模块自身噪声太大，微弱的输入信号会被噪声盖住，就像在嘈杂的车间里听不清别人说话。

噪声系数就是衡量"模块自身有多吵"的指标，**越小越好**（更安静）。

| 参数 | 指标 | 通俗解释 |
|------|:---:|------|
| 最大噪声系数 | ≤ 1.30 dB | 在所有频率中，最吵的那个频点，噪声不能超过这个数 |
| 平均噪声系数 | < 1.20 dB | 所有频率的噪声平均下来，要低于这个数 |

测试覆盖频点：0.95 / 1.00 / 1.05 / 1.10 / 1.15 / 1.20 / 1.25 / 1.30 / 1.35 / 1.40 / 1.45 GHz

### 5.2 增益

**增益**就是模块把信号放大了多少。比如输入 −50 dBm，输出 0 dBm，增益就是 50 dB。增益越大，模块放大能力越强。

| 参数 | 指标 | 通俗解释 |
|------|:---:|------|
| 平均增益 | > 50.0 dB | 所有频率的增益平均下来，要高于这个值 |
| 增益平坦度 | < 2.50 dB | 不同频率的增益应该差不多。如果 1GHz 放大 52dB、1.4GHz 只放大 49dB，差值 3dB 就超了——信号在不同频率"音量"不一致 |

### 5.3 相位噪声

**相位噪声**衡量模块内部振荡器的"纯净度"。

想象你敲一个音叉，它会发出一个纯音。但如果你的手在抖，音叉的声音就会忽高忽低。相位噪声就是这种"抖动"的量度——抖得越厉害，相位噪声越差。

数值是**负数**（dBc/Hz），**越负越好**（抖动越小）。

测试条件：中心频率 1.2 GHz。

| 偏移频率 | 指标 | 通俗解释 |
|----------|:---:|------|
| 偏离载波 100 Hz | < −65.0 dBc/Hz | 距离信号中心 100Hz 处，噪声比信号低至少 65dB |
| 偏离载波 1 KHz | < −74.0 dBc/Hz | 距离 1KHz 处，低至少 74dB |
| 偏离载波 10 KHz | < −80.0 dBc/Hz | 距离 10KHz 处，低至少 80dB |
| 偏离载波 100 KHz | < −95.0 dBc/Hz | 距离 100KHz 处，低至少 95dB |

## 六、发射通道性能参数

### 6.1 输出功率

模块发射时能输出多大的信号功率。功率越大，信号传得越远。

测试条件：TX IF 输入 −14 dBm（约 0.04 毫瓦），RF OUT 输出经衰减器和线缆损耗补偿后测量。

> −14 dBm 是什么意思？dBm 是射频功率单位。0 dBm = 1 毫瓦，每减 10 dBm 功率变为十分之一。−14 dBm ≈ 0.04 毫瓦，是很小的信号。模块把它放大到 32.8 dBm ≈ 2 瓦，放大了约 5 万倍。

| IF 输入频率 | RF 输出频率 | 输出功率 |
|:---:|:---:|:---:|
| 1050 MHz | 5.95 GHz | ≥ 32.80 dBm（约 1.9 瓦） |
| 1200 MHz | 6.10 GHz | ≥ 32.80 dBm |
| 1550 MHz | 6.45 GHz | ≥ 32.80 dBm |

> 三个频率的输出功率限值可以不同。例如要求 1050MHz  ≥ 34.5 dBm，1550MHz ≥ 34.1 dBm。

### 6.2 增益

增益 = 输出功率 − 输入功率。输入固定为 −14 dBm，输出 ≥ 32.80 dBm，所以增益 ≥ 32.80 − (−14) = 46.80 dB。

**增益不需要单独关注**——输出功率合格了，增益自然合格（因为输入功率是固定的）。这是自动换算的。

### 6.3 发射平坦度

IF 输入从 0.95 GHz 到 1.55 GHz 连续变化（扫频），观测 RF OUT 输出在 5.85~6.45 GHz 整个范围内的功率变化。

| 参数 | 指标 | 通俗解释 |
|------|:---:|------|
| 带内平坦度 | < 3.00 dB | 整个频段内，最强输出和最弱输出的差距不超过 3dB |

### 6.4 相位噪声

测试条件：IF 输入 1.2 GHz / −20 dBm，RF 中心 6.1 GHz。

| 偏移频率 | 指标 |
|----------|:---:|
| 偏离载波 100 Hz | < −65.0 dBc/Hz |
| 偏离载波 1 KHz | < −74.0 dBc/Hz |
| 偏离载波 10 KHz | < −80.0 dBc/Hz |
| 偏离载波 100 KHz | < −95.0 dBc/Hz |

> 物理含义与接收通道相同，参见 5.3 节。

## 七、收发同时工作时的相互影响

接收通道在收微弱信号的时候，如果旁边的发射通道正在大功率发射，发射信号可能会"泄露"到接收通道里。这就像你在听一个人小声说话，旁边有个人在大喊大叫——你就听不清了。

这种泄露会导致接收通道的**噪底抬高**。噪底是接收通道本身的背景噪声水平。噪底抬高 = 灵敏度变差 = 本来能收到的微弱信号收不到了。

| 发射 IF 频率 | 同时监测的接收 IF 频率 | 噪底抬高允许上限 |
|:---:|:---:|:---:|
| 1050 MHz | 1300 MHz | ≤ 2.00 dB |
| 1200 MHz | 950 MHz | ≤ 2.00 dB |

> 两个频率对分别测试。例如 TX 在 1050MHz 发射时，同时测 RX 在 1300MHz 的噪底，看看比发射关闭时抬高了几个 dB。

## 八、使用注意事项

1. **RX 和 TX 口供电电压不同，绝对不能接反。** RX IF 口 12V，TX IF 口 24V。把 24V 接到 RX 口会烧坏模块。
2. **先接好所有线缆，最后再通电。** 不要带电插拔 SMA 接头。
3. **10 MHz 参考信号必须先给。** 模块内部的频率源没有参考信号时无法锁定，输出频率会不准。参考信号功率建议 −5 ~ +5 dBm。
4. **RF OUT 输出功率可达 2W 以上。** 不要把 RF OUT 直接连到频谱仪或其他仪器的输入口——会烧坏仪器输入电路。必须通过衰减器（建议 30dB 以上）再连接。
5. **IF 口必须配合 Bias-Tee 使用。** 同轴电缆里的 DC 电和 RF 信号混在一起，需要 Bias-Tee 把它们分开。Bias-Tee 的 RF+DC 口接模块、RF 口接信号设备、DC 口接电源。
6. **防静电。** 模块内部的半导体器件对静电敏感。操作前摸一下接地金属释放身体静电，最好佩戴防静电手环。
