## 四、指标参考范围

### 4.1 RX 噪声系数 + 增益

| 指标 | 限值 | 说明 |
|------|------|------|
| NF 最大值 | ≤ 1.30 dB | 11 个频点中噪声系数最大值 |
| NF 平均值 | < 1.20 dB | 11 个频点噪声系数算术平均 |
| 增益平均值 | > 50.0 dB | 11 个频点增益算术平均 |
| 增益平坦度 | < 2.50 dB | 去除首尾频点后相邻增益差的最大值 |

**测试频点**：0.95, 1.00, 1.05, 1.10, 1.15, 1.20, 1.25, 1.30, 1.35, 1.40, 1.45 GHz

**测试条件**：
- 频谱仪模式：NF（噪声系数模式）
- RX 电源：开启
- TX 电源：关闭
- 开关配置：[1, 0, 0, 0]

### 4.2 RX 相位噪声

| 偏移频率 | 限值 |
|----------|------|
| 100 Hz | < −65.0 dBc/Hz |
| 1 KHz | < −74.0 dBc/Hz |
| 10 KHz | < −80.0 dBc/Hz |
| 100 KHz | < −95.0 dBc/Hz |

**测试条件**：
- 频谱仪模式：PN（相位噪声模式）
- 中心频率：1.2 GHz
- VSG：3950 MHz，−20 dBm CW
- RX 电源：开启
- TX 电源：关闭
- 开关配置：[0, 0, 0, 0]

### 4.3 TX 增益 + 输出功率

| 频点 (IF) | RF 频率 | Pout 限值 | Gain 限值 |
|-----------|---------|-----------|-----------|
| 1050 MHz | 5.95 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |
| 1200 MHz | 6.10 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |
| 1550 MHz | 6.45 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |

> **注**：Pout 限值可按频点独立配置；Gain 限值 = Pout 限值 − VSG 功率（自动计算）。以上为典型值，实际以配置文件为准。

**测试条件**：
- 频谱仪模式：SA
- VSG：各频点 CW，−14 dBm
- RX 电源：关闭
- TX 电源：开启
- 开关配置：[1, 1, 1, 0]
- RF 线损补偿已计入

### 4.4 TX 平坦度 + 相位噪声

**平坦度**：

| 指标 | 限值 |
|------|------|
| 发射平坦度 | < 3.00 dB |

**相位噪声**：

| 偏移频率 | 限值 |
|----------|------|
| 100 Hz | < −65.0 dBc/Hz |
| 1 KHz | < −74.0 dBc/Hz |
| 10 KHz | < −80.0 dBc/Hz |
| 100 KHz | < −95.0 dBc/Hz |

**测试条件（平坦度）**：
- 频谱仪模式：SA，Max Hold
- VSG：扫频 0.95~1.55 GHz，步进 1 MHz，驻留 20 ms，−14 dBm
- RF 范围：5.85~6.45 GHz
- Max Hold 等待：20 秒
- RX 电源：关闭
- TX 电源：开启
- 开关配置：[1, 1, 1, 0]

**测试条件（相位噪声）**：
- 频谱仪模式：PN
- 中心频率：6.1 GHz
- VSG：1200 MHz，−20 dBm CW

### 4.5 TX-RX 收发干扰

| 指标 | 限值 |
|------|------|
| 噪底差异最大值 | ≤ 2.00 dB |

**测试频率对**：

| TX IF | RX IF | 说明 |
|-------|-------|------|
| 1050 MHz | 1300 MHz | TX 发射时监测 RX 频段噪底变化 |
| 1200 MHz | 950 MHz | 同上 |

**测试条件**：
- 频谱仪模式：SA，噪声 Marker
- VSG：CW，−14 dBm
- RX 电源：开启
- TX 电源：TXOFF 时关 / TXON 时开
- TX 开启后等待 10 秒稳定
- 开关配置：[1, 0, 0, 0]
- POS 检波器（噪声 Marker 要求）

## 五、同轴供电说明

```
┌─────────────┐         ┌──────────────────┐
│  仪器系统    │ IF 端口 │  C波段射频模块    │
│             │◄───────►│  (C10F004B)      │
│  SMU200A    │ 同轴电缆 │                  │
│  N9020A     │         │  IF ◄────────► RF │
│  TDK电源×2  │         │                  │
│  UDC-0624F  │         └──────────────────┘
└─────────────┘
```

- **DC 供电**通过 IF 端口同轴馈入，由开关矩阵控制 RX/TX 电源通断
- **10 MHz 参考信号**由信号源提供，通过 IF 端口送入模块作为本振参考
- **IF 信号**双向传输：发射时 IF → 模块 → RF 输出；接收时 RF → 模块 → IF 输出
- 开关矩阵负责切换 RX/TX 通路以及 IF/RF 信号流向

## 六、测试流程

### 6.1 准备工作

1. 确认所有仪表已上电、网络连通
2. 确认 `user_settings.json` 中 IP 地址与实际一致
3. 将被测模块 IF 端口连接至开关矩阵
4. 将被测模块 RF 端口连接至开关矩阵（经衰减器）
5. 输入正确的序列号

### 6.2 运行测试

1. 点击 **「连接全部仪表」** → 确认 5 项全部绿色
2. 点击 **「运行全部测试」** 或单独点击测试按钮
3. 观察结果卡片：绿色 ✓ PASS / 红色 ✗ FAIL
4. 点击卡片可查看详细数据表格
5. 测试完成后点击 **「写入报告」** 生成 TXT + DOCX 报告

### 6.3 换下一个模块

1. 点击 **「清空结果」** 清除上一个模块的测试状态
2. 输入新序列号（或点击写入报告后自动递增）
3. 更换模块，重复 6.2

## 八、常见问题

| 现象 | 可能原因 | 排除方法 |
|------|----------|----------|
| 频谱仪连接失败 | IP 配置错误或网络断开 | 检查 `user_settings.json` 中频谱仪 IP，ping 测试 |
| NF 测量异常偏高 | NF 模板未校准或线缆损耗大 | 重新校准频谱仪 NF 模式，检查线缆连接 |
| TX 功率偏低 | 衰减器损耗未补偿或模块供电不足 | 检查 RF 线损配置，确认 TX 电源电压 |
| 收发干扰超标 | 模块屏蔽不良或开关隔离度不足 | 检查模块接地和屏蔽，确认开关矩阵状态 |
| 截图保存失败 | 仪器内部路径配置错误 | 确认 `screenshot.instrument_internal_path` 为 N9020A 内部有效路径 |
| 按钮灰色不可点 | 结果面板未刷新 | 运行至少一项测试即可激活 |
