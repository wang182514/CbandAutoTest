> 版本：1.0  
> 适用系统：CBandAutoTest (Python/PySide6)

---

## 一、被测件概述

| 项目 | 说明 |
|------|------|
| 产品名称 | C波段射频模块 |
| 产品型号 | C10F004B |
| 供电方式 | **同轴供电** — 电源、10MHz 参考信号、IF 信号均从 IF 端口馈入 |
| IF 端口功能 | DC 供电 + 10MHz 参考 + IF 信号（发射/接收双向） |
| RF 端口 | C 波段射频输出（TX）/ 输入（RX） |

### 同轴供电示意

```
仪器系统 ── 同轴电缆 ── IF 端口 ── C波段射频模块 ── RF 端口
              ▲                            │
              │   DC 供电                   │
              │   10MHz 参考                │   C 波段信号
              │   IF 信号                   │   (5.85~6.45 GHz)
              │                            │
              └────────────────────────────┘
```

- **DC 供电**通过 IF 端口同轴馈入
- **10 MHz 参考信号**由信号源提供，通过 IF 端口送入模块作为本振参考
- **IF 信号**双向传输：发射时 IF → 模块 → RF 输出；接收时 RF → 模块 → IF 输出

## 四、测试细则

### 4.1 RX 噪声系数 + 增益

**测试频点**：0.95, 1.00, 1.05, 1.10, 1.15, 1.20, 1.25, 1.30, 1.35, 1.40, 1.45 GHz（共 11 点）

| 指标 | 参考范围 | 说明 |
|------|----------|------|
| NF 最大值 | ≤ 1.30 dB | 11 个频点中噪声系数最大值 |
| NF 平均值 | < 1.20 dB | 11 个频点噪声系数算术平均 |
| 增益平均值 | > 50.0 dB | 11 个频点增益算术平均 |
| 增益平坦度 | < 2.50 dB | 去除首尾频点后相邻增益差的最大值 |

**测试步骤**：
1. 频谱仪切换至 NF（噪声系数）模式，加载预校准模板
2. RX 电源开启，TX 电源关闭
3. 频谱仪单次触发测量，同步读取全部频点的 NF 和 Gain
4. 计算统计值，判定合格性

---

### 4.2 RX 相位噪声

| 偏移频率 | 参考范围 |
|----------|----------|
| 100 Hz | < −65.0 dBc/Hz |
| 1 KHz | < −74.0 dBc/Hz |
| 10 KHz | < −80.0 dBc/Hz |
| 100 KHz | < −95.0 dBc/Hz |

**测试步骤**：
1. 频谱仪切换至 PN（相位噪声）模式，加载预校准模板
2. 设置中心频率 1.2 GHz
3. 信号源输出 3950 MHz / −20 dBm CW
4. RX 电源开启，TX 电源关闭
5. 频谱仪单次触发测量，读取 4 个偏移点的相位噪声值

---

### 4.3 TX 增益 + 输出功率

| 频点 (IF) | 对应 RF 频率 | Pout 参考范围 | Gain 参考范围 |
|-----------|-------------|--------------|--------------|
| 1050 MHz | 5.95 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |
| 1200 MHz | 6.10 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |
| 1550 MHz | 6.45 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |

> Gain 限值 = Pout 限值 − VSG 输入功率（自动换算），以上为 VSG = −14 dBm 时的典型值。

**测试步骤**：
1. 频谱仪切换至 SA 模式，加载预校准模板
2. TX 电源开启，RX 电源关闭
3. 信号源输出 CW，依次设置 1050 / 1200 / 1550 MHz，−14 dBm
4. 频谱仪在每个 RF 频点执行峰值搜索，读取 Pout
5. Gain = Pout − VSG 功率（含 RF 线损补偿）
6. 逐频点判定合格性

---

### 4.4 TX 平坦度 + 相位噪声

**平坦度**：

| 指标 | 参考范围 |
|------|----------|
| 发射平坦度 | < 3.00 dB |

**测试步骤（平坦度）**：
1. 频谱仪 SA 模式，Max Hold 轨迹
2. 信号源设置为扫频：IF 0.95~1.55 GHz，步进 1 MHz，−14 dBm
3. RF 观测范围：5.85~6.45 GHz
4. 等待 20 秒 Max Hold 稳定后，读取峰-峰值即为平坦度

**相位噪声**：

| 偏移频率 | 参考范围 |
|----------|----------|
| 100 Hz | < −65.0 dBc/Hz |
| 1 KHz | < −74.0 dBc/Hz |
| 10 KHz | < −80.0 dBc/Hz |
| 100 KHz | < −95.0 dBc/Hz |

**测试步骤（相位噪声）**：
1. 频谱仪 PN 模式，中心频率 6.1 GHz
2. 信号源 1200 MHz / −20 dBm CW
3. TX 电源开启，RX 电源关闭
4. 单次触发测量，读取 4 个偏移点

---

### 4.5 TX-RX 收发干扰

| 指标 | 参考范围 |
|------|----------|
| 噪底差异最大值 | ≤ 2.00 dB |

**测试频率对**：

| 轮次 | TX IF（发射） | RX IF（接收监测） |
|:---:|-------------|-----------------|
| 1 | 1050 MHz | 1300 MHz |
| 2 | 1200 MHz | 950 MHz |

**测试步骤**：
1. RX 电源开启，TX 电源关闭
2. 信号源输出对应 TX IF 频率 CW，−14 dBm
3. 频谱仪 SA 模式，以 RX IF 频率为中心、±10 MHz 范围内，用噪声 Marker 读取噪底（TXOFF 基准值）
4. TX 电源开启，等待 10 秒稳定后，再次读取噪底（TXON 值）
5. 计算 TXON − TXOFF 噪底差异
6. 在两组频率对上重复，取差异最大值

## 五、操作流程

### 5.1 测试前准备

1. 确认所有仪表已上电、网络连通
2. 将被测模块 IF 端口连接测试系统同轴接口
3. 将被测模块 RF 端口连接测试系统 RF 接口（经衰减器）
4. 输入正确的序列号

### 5.2 执行测试

1. 连接全部仪表，确认连接状态正常
2. 选择「运行全部测试」或单独选择测试项
3. 观察结果：绿色 PASS 表示合格，红色 FAIL 表示不合格
4. 点击测试卡片可查看详细测量数据
5. 测试完成后生成报告

### 5.3 测试结束

1. 断开仪表连接
2. 更换下一个模块
