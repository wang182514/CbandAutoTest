# C10F004B C波段射频模块 — 产品规格书

> 版本：1.0

---

## 一、产品概述

C10F004B 是一款 C 波段射频收发模块，用于将 L 波段 IF 信号变频至 C 波段 RF 信号（发射），以及将 C 波段 RF 信号变频至 L 波段 IF 信号（接收）。

| 项目 | 说明 |
|------|------|
| 产品名称 | C波段射频模块 |
| 产品型号 | C10F004B |
| 供电方式 | **同轴供电** — DC 电源、10MHz 参考信号、IF 信号共用 IF 端口馈入 |
| IF 频率范围 | 0.95 ~ 1.55 GHz |
| RF 频率范围 | 5.85 ~ 6.45 GHz |

---

## 二、接口定义

```
                ┌──────────────────┐
   IF 端口 ◄───►│  C10F004B        │◄───► RF 端口
  (同轴, SMA)   │  C波段射频模块    │      (同轴, SMA)
                └──────────────────┘
```

### IF 端口（同轴复合接口）

通过单根同轴电缆同时承载以下信号：

| 信号 | 说明 |
|------|------|
| DC 供电 | 模块工作电源，通过 Bias-Tee 馈入 |
| 10 MHz 参考 | 外部本振参考信号，锁相模块内部频率源 |
| IF 信号 | L 波段中频信号，双向传输（发射/接收） |

### RF 端口

C 波段射频信号输入/输出端口。

## 三、射频性能参数

### 3.1 接收通道（RX）

#### 噪声系数 + 增益

| 参数 | 规格 | 测试条件 |
|------|:---:|------|
| 噪声系数（最大值） | ≤ 1.30 dB | IF = 0.95~1.45 GHz，11 频点中取最大值 |
| 噪声系数（平均值） | < 1.20 dB | 同上，11 频点算术平均 |
| 增益（平均值） | > 50.0 dB | 同上 |
| 增益平坦度 | < 2.50 dB | 去除首尾频点后相邻增益差最大值 |

测试频点：0.95 / 1.00 / 1.05 / 1.10 / 1.15 / 1.20 / 1.25 / 1.30 / 1.35 / 1.40 / 1.45 GHz

#### 相位噪声

中心频率 1.2 GHz，VSG = 3950 MHz / −20 dBm。

| 偏移频率 | 规格 |
|----------|:---:|
| 100 Hz | < −65.0 dBc/Hz |
| 1 KHz | < −74.0 dBc/Hz |
| 10 KHz | < −80.0 dBc/Hz |
| 100 KHz | < −95.0 dBc/Hz |

---

### 3.2 发射通道（TX）

#### 增益 + 输出功率

VSG 输入 −14 dBm，RF 端口输出（含路径损耗补偿）：

| IF 频率 | RF 频率 | 输出功率 | 增益 |
|---------|---------|:---:|:---:|
| 1050 MHz | 5.95 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |
| 1200 MHz | 6.10 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |
| 1550 MHz | 6.45 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |

> Gain = Pout − VSG 输入功率。以上为典型配置值，可逐频点独立设定限值。

#### 发射平坦度

IF 扫频 0.95~1.55 GHz，RF 观测 5.85~6.45 GHz：

| 参数 | 规格 |
|------|:---:|
| 带内平坦度 | < 3.00 dB |

#### 相位噪声

中心频率 6.1 GHz，VSG = 1200 MHz / −20 dBm。

| 偏移频率 | 规格 |
|----------|:---:|
| 100 Hz | < −65.0 dBc/Hz |
| 1 KHz | < −74.0 dBc/Hz |
| 10 KHz | < −80.0 dBc/Hz |
| 100 KHz | < −95.0 dBc/Hz |

| TX IF 频率 | RX IF 监测频率 | 噪底差异限值 |

| 1050 MHz | 1300 MHz | ≤ 2.00 dB |

## 五、注意事项

1. IF 端口为同轴复合接口，需配合 Bias-Tee 使用以分离 DC 与 RF 信号
2. 上电前确认 DC 供电电压与极性符合要求
3. RF 端口输出功率较高，测试及使用时需接入适当衰减器
4. 10 MHz 参考信号需稳定且功率适中（典型 −5 ~ +5 dBm），否则影响相位噪声性能
5. 模块对静电敏感，操作时需采取防静电措施
