# C10F004B C波段射频模块 — 产品规格书

> 版本：1.0

---

## 一、产品概述

C10F004B 是一款 C 波段射频收发变频模块，实现 L 波段 IF 信号与 C 波段 RF 信号之间的双向频率转换。

本系列含两个子型号：C10F004B1 / C10F004B2

除发射功率及对应工作电流不同外，两个型号接口定义、频率范围和其他射频指标均一致。

| 项目       | 说明                                       |
| -------- | ---------------------------------------- |
| 产品名称     | C波段射频模块                                  |
| 产品型号     | C10F004B1 / C10F004B2                    |
| TX IF 频率 | 0.95 ~ 1.55 GHz                          |
| TX RF 频率 | 5.85 ~ 6.45 GHz                          |
| RX RF 频率 | 3.70 ~ 4.20 GHz                          |
| RX IF 频率 | 0.95 ~ 1.45 GHz                          |
| 供电方式     | 同轴供电（DC + 10MHz 参考 + IF 信号经由合路器进入 IF 端口） |

---

## 二、接口定义

模块共 4 个 SMA 接口，分接收和发射两组：

| 接口     | 刻字                  | 方向  | 说明                                     |
| ------ | ------------------- |:---:| -------------------------------------- |
| RX IF  | `IF IN / 12V / 10M` | 输出  | 接收中频输出。承载 IF 信号、12V DC 供电、10MHz 参考。    |
| RF IN  | `RF IN`             | 输入  | 接收射频输入。C 波段信号（3.70~4.20 GHz）由此进入模块。    |
| TX IF  | `IF IN / 24V / 10M` | 输入  | 发射中频输入。承载 IF 信号、24V DC 供电、10MHz 参考。    |
| RF OUT | `RF OUT`            | 输出  | 发射射频输出。变频后的 C 波段信号（5.85~6.45 GHz）由此输出。 |

**信号流向**：

```
接收:  RF IN ──→ 模块 ──→ RX IF   (3.70~4.20 GHz 进 → 0.95~1.45 GHz 出)
发射:  TX IF ──→ 模块 ──→ RF OUT  (0.95~1.55 GHz 进 → 5.85~6.45 GHz 出)
```

### 同轴供电说明

RX IF 和 TX IF 端口均为同轴复合接口，单根同轴电缆同时传输三种信号：

| 信号        | 用途                      |
| --------- | ----------------------- |
| DC 电源     | 模块工作电源（RX 12V / TX 24V） |
| 10 MHz 参考 | 外部参考时钟，用于锁定模块内部频率源      |
| IF 信号     | 中频信号                    |

DC 电源和 10 MHz 参考信号通过**合路器**合并后，与 IF 信号一同馈入 IF 端口。

## 四、电气特性

### 4.1 供电电压

| 端口    | 电压     | 说明     |
| ----- |:------:| ------ |
| RX IF | 12V DC | 接收通道供电 |
| TX IF | 24V DC | 发射通道供电 |

### 4.2 工作电流

#### C10F004B1（2W 版本）

| 通道  | 状态               | 电流            |
| --- | ---------------- |:-------------:|
| 接收  | 上电 / 工作（IF 信号开启） | 0.15 ~ 0.25 A |
| 发射  | 上电（IF 信号关闭）      | 0.15 ~ 0.25 A |
| 发射  | 工作（IF 信号开启）      | 0.45 ~ 0.70 A |

#### C10F004B2（3W 版本）

| 通道  | 状态               | 电流            |
| --- | ---------------- |:-------------:|
| 接收  | 上电 / 工作（IF 信号开启） | 0.15 ~ 0.25 A |
| 发射  | 上电（IF 信号关闭）      | 0.40 ~ 0.55 A |
| 发射  | 工作（IF 信号开启）      | 0.60 ~ 0.90 A |

> 发射工作电流随 IF 输入频率升高而升高，以上为全频段范围。

### 4.3 参考信号

DC 供电与 10 MHz 参考信号经合路器合并后馈入 IF 端口。10 MHz 参考信号建议功率 −5 ~ +5 dBm。模块内部频率源需外部参考锁定，无参考信号时组件无法正常工作。收发 10M 锁定时蓝色指示灯常亮，异常状态对照：

| 指示灯状态 | 含义 |
|-----------|------|
| 亮一秒灭一秒 | 收发 10M 均不锁定 |
| 亮三秒灭一秒 | 发射 10M 不锁定 |
| 亮一秒灭三秒 | 接收 10M 不锁定 |

## 五、使用注意事项

1. RX IF（12V）与 TX IF（24V）供电电压不同，**接反会损坏模块**。
2. 上电前确认所有线缆连接正确，先接信号线后通电。
3. 10 MHz 参考信号应先于 DC 供电或同时接入，确保频率源正常锁定。
4. RF OUT 端口输出功率较高（B1 约 2W / B2 约 3W），连接测试设备时需接入衰减器（建议 ≥ 30 dB）。
