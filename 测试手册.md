# C10F004B C波段射频模块 — 产品规格书

> 版本：1.0

---

## 一、产品概述

C10F004B 是一款 C 波段射频收发变频模块，实现 L 波段 IF 信号与 C 波段 RF 信号之间的双向频率转换。

| 项目 | 说明 |
|------|------|
| 产品名称 | C波段射频模块 |
| 产品型号 | C10F004B |
| IF 频率 | RX: 0.95 ~ 1.45 GHz / TX: 0.95 ~ 1.55 GHz |
| RF 频率 | 5.85 ~ 6.45 GHz |
| 供电方式 | 同轴供电（DC + 10MHz 参考 + IF 信号复用 IF 端口） |

---

## 二、接口定义

模块共 4 个 SMA 同轴接口，分接收和发射两组：

| 接口 | 刻字 | 方向 | 说明 |
|------|------|:---:|------|
| RX IF | `IF IN / 12V / 10M` | 输入 | 接收中频输入。承载 IF 信号、12V DC 供电、10MHz 参考。 |
| RF IN | `RF IN` | 输入 | 接收射频输入。C 波段信号由此进入模块。 |
| TX IF | `IF IN / 24V / 10M` | 输入 | 发射中频输入。承载 IF 信号、24V DC 供电、10MHz 参考。 |
| RF OUT | `RF OUT` | 输出 | 发射射频输出。变频后的 C 波段信号由此输出。 |

**信号流向**：

```
接收:  RF IN ──→ 模块 ──→ RX IF   (C波段进 → L波段出)
发射:  TX IF ──→ 模块 ──→ RF OUT  (L波段进 → C波段出)
```

### 同轴供电说明

RX IF 和 TX IF 端口均为同轴复合接口，单根同轴电缆同时传输三种信号：

| 信号 | 用途 |
|------|------|
| DC 电源 | 模块工作电源（RX 12V / TX 24V） |
| 10 MHz 参考 | 外部参考时钟，用于锁定模块内部频率源 |
| IF 信号 | L 波段中频信号 |

使用时需配合 Bias-Tee 分离 DC 与 RF 信号。Bias-Tee 的 RF+DC 端接模块 IF 口，RF 端接信号设备，DC 端接电源。

### 3.1 接收通道

#### 噪声系数与增益

测试频点：0.95 / 1.00 / 1.05 / 1.10 / 1.15 / 1.20 / 1.25 / 1.30 / 1.35 / 1.40 / 1.45 GHz

| 参数 | 规格 |
|------|:---:|
| 噪声系数最大值 | ≤ 1.30 dB |
| 噪声系数平均值 | < 1.20 dB |
| 增益平均值 | > 50.0 dB |
| 增益平坦度 | < 2.50 dB |

> 增益平坦度定义为去除首尾频点后，相邻频点间增益差的最大绝对值。

#### 相位噪声

| 1 KHz | < −74.0 dBc/Hz |
| 10 KHz | < −80.0 dBc/Hz |

### 3.2 发射通道

#### 输出功率与增益

IF 输入 −14 dBm，RF 输出含线损补偿。

| IF 频率 | RF 频率 | 输出功率 | 增益 |
|---------|---------|:---:|:---:|
| 1050 MHz | 5.95 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |
| 1200 MHz | 6.10 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |
| 1550 MHz | 6.45 GHz | ≥ 32.80 dBm | ≥ 46.80 dB |

> 各频点输出功率限值可独立配置。增益 = 输出功率 − IF 输入功率，自动换算。

#### 发射平坦度

IF 0.95~1.55 GHz 扫频，RF 5.85~6.45 GHz 观测。

| 参数 | 规格 |
|------|:---:|
| 带内平坦度 | < 3.00 dB |

#### 相位噪声

中心频率 6.1 GHz。

| 偏移频率 | 规格 |
|----------|:---:|
| 100 Hz | < −65.0 dBc/Hz |
| 1 KHz | < −74.0 dBc/Hz |

| TX IF 频率 | 监测 RX IF 频率 | 噪底差异上限 |
|:---:|:---:|:---:|
| 1050 MHz | 1300 MHz | ≤ 2.00 dB |
| 1200 MHz | 950 MHz | ≤ 2.00 dB |

## 四、供电与参考信号要求

| 端口 | 电压 | 说明 |
|------|:---:|------|
| RX IF | 12V DC | 接收通道供电，不可接 24V |
| TX IF | 24V DC | 发射通道供电，不可接 12V |

10 MHz 参考信号通过 Bias-Tee 馈入 IF 口，建议功率 −5 ~ +5 dBm。模块内部频率源需外部参考锁定，无参考信号时输出频率可能偏移。

---

## 五、使用注意事项

1. RX IF（12V）与 TX IF（24V）供电电压不同，**接反会损坏模块**。
2. 上电前确认所有线缆连接正确，先接信号线后通电。
3. 10 MHz 参考信号应先于 DC 供电或同时接入，确保频率源正常锁定。
4. RF OUT 端口输出功率较高（约 2W），连接测试设备时需接入衰减器（建议 ≥ 30 dB）。
5. IF 端口为同轴复合接口，必须通过 Bias-Tee 分离 DC 与 RF 信号。
6. 模块含静电敏感器件，操作时请采取防静电措施。
